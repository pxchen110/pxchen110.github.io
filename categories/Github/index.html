з”ЁGithub+Hexoе»әз«ӢеҖӢдәәйғЁиҗҪж јпјҡж–№жі•иҲҮе•ҸйЎҢзҙҖйҢ„

 2024е№ҙ2жңҲ1ж—Ҙ


Photo by Lukas Blazek on Unsplash

жұәе®ҡж’°еҜ«еҖӢдәәйғЁиҗҪж јеҫҢпјҢеҸғиҖғиЁұеӨҡз¶ІеҸӢзҡ„е№іеҸ°жҺЁи–ҰпјҢжңҖзөӮжұәе®ҡйҒёж“Үе…Қд»ҳиІ»е’Ңе®ўиЈҪеҢ–иҮӘз”ұеәҰй«ҳзҡ„GithubпјҢеҗҢжҷӮз·ҙзҝ’ markdownиӘһжі•, git з®ЎзҗҶйғЁеҲҶгҖӮжң¬зҜҮж–Үз« иЁҳйҢ„жҲ‘е»әз«Ӣз¶Із«ҷзҡ„жӯҘй©ҹе’ҢйҒҮеҲ°зҡ„е•ҸйЎҢпјҢйҒҝеқ‘йҒҝйӣ·дҪ жҲ‘жңүиІ¬гҖӮ

HTML+CSSжҲҗжһңеұ•(1)пјҡеҸҜж„ӣиІ“е’Әз¶Із«ҷ

 2024е№ҙ2жңҲ8ж—Ҙ

еңЁudemyдёҠеӯёдәҶhtml+cssзҡ„зҹҘиӯҳпјҢе…үзңӢдёҚжңғйҖІжӯҘпјҢеӢ•жүӢеҒҡдёҖйҒҚжүҚзҹҘйҒ“дёҚзҶҹзҡ„ең°ж–№пјҢжҲ‘еҜ«дёҖеҖӢжЁҷйЎҢзӮәиҰҒдҫҶжҲ‘е®¶зңӢиІ“е—Һзҡ„е°Ҹз¶Ій Ғ

HTML+CSSжҲҗжһңеұ•(2)пјҡд»ҝеҜ«жӢјеӨҡеӨҡз¶Із«ҷ

 2024е№ҙ2жңҲ11ж—Ҙ

еҸғиҖғиіҮж–ҷдҫҶжәҗпјҡudemy иӘІзЁӢ

еҫһдёҠж¬Ўз·ҙзҝ’еҫһйӣ¶еүөйҖ з¶Ій ҒдҪҲеұҖеҫҢпјҢзҷјзҸҫиҮӘе·ұзҡ„жҠҖе·§е’ҢзҫҺж„ҹйғҪжңүеҫ…йҖІжӯҘпјҢжұәе®ҡи·ҹи‘—udemyиӘІзЁӢзҡ„еҜҰдҪңй …зӣ®:жҗӯе»әжӢјеӨҡеӨҡз¶Із«ҷйҰ–й ҒпјҢдҫҶеӯёзҝ’зңҹеҜҰз¶Із«ҷжңүе“Әдәӣеёёз”Ёзҡ„дҪҲеұҖе’ҢжҠҖе·§гҖӮ

PythonеҜҰз”Ёзі»еҲ—пјҡ smtplib жЁЎзө„еҜҰзҸҫиҮӘеӢ•еҜ„дҝЎ

 2024е№ҙ2жңҲ15ж—Ҙ


Photo by Brett Jordan on Unsplash

еңЁдҝЎз®ұж”¶д»¶еӨҫпјҢдёҚд№ҸзңӢеҲ°иЁұеӨҡиҮӘеӢ•еҜ„еҮәзҡ„дҝЎд»¶пјҢеғҸжҳҜдҫҶиҮӘ slurm job йҖҡзҹҘеҲҶжһҗзӢҖж…Ӣзҡ„дҝЎпјҢд»ҘеҸҠиЁӮй–ұе…§е®№зҡ„иҮӘеӢ•йҖҡзҹҘзӯүгҖӮиҮӘеӢ•еҜ„дҝЎдёҚеғ…зҜҖзңҒдәәеҠӣж“ҚдҪңпјҢд№ҹеҸҜе»ЈзҷјдҝЎд»¶еўһеҠ еӮіж’ӯж•ҲзҺҮгҖӮд»ҠеӨ©е°ұиҰҒд»Ӣзҙ№еҰӮдҪ•з”ЁpythonеҜҰзҸҫиҮӘеӢ•еҜ„дҝЎ

PythonеҜҰз”Ёзі»еҲ—пјҡ дёүзЁ®еј•е…ҘеӨ–йғЁеҸғж•ёзҡ„ж–№жі•

 2024е№ҙ2жңҲ21ж—Ҙ


Photo by Artturi Jalli on Unsplash

ж’°еҜ«python scriptжҷӮпјҢи®Ҡж•ёе…§е®№иӢҘиғҪйҡЁдҪҝз”ЁиҖ…еҪҲжҖ§жӣҙж”№пјҢеҸҜжҸҗеҚҮзЁӢејҸеҜҰз”ЁжҖ§пјӣжң¬ж–Үд»Ӣзҙ№дёүзЁ®еёёз”Ёзҡ„еј•е…ҘеӨ–йғЁеҸғж•ёжЁЎзө„: sys, argparse, д»ҘеҸҠ clickпјҢж–Үз« е°Үд»Ӣзҙ№жЁЎзө„/еҘ—д»¶зҡ„дҪҝз”Ёж–№ејҸпјҢж’°еҜ«зӣёеҗҢијёеҮәзҡ„еҜҰдҫӢпјҢдёҰеҲҶжһҗе„Әзјәй»һд»Ҙдҫӣи®ҖиҖ…еҸғиҖғгҖӮ

PythonеҜҰз”Ёзі»еҲ—пјҡ subprocessжЁЎзө„з®ЎзҗҶеӯҗйҖІзЁӢ

 2024е№ҙ2жңҲ22ж—Ҙ


Photo by Artturi Jalli on Unsplash

ж’°еҜ«зЁӢејҸе°ҲжЎҲжҷӮпјҢеёёйңҖиҰҒеңЁзҲ¶йҖІзЁӢдёӢеүөе»әеӯҗйҖІзЁӢйҖІиЎҢж”Ҝз·ҡд»»еӢҷпјҢдҫӢеҰӮеңЁshell script(.sh)дёӯпјҢжңғеңЁ linux command зҡ„еҹәзӨҺдёӢпјҢеҢ…еҗ« python script, R script зҡ„еҹ·иЎҢпјҢйҖҸйҒҺдёІиҒҜијёеҮә\ијёе…ҘжӘ”жЎҲзҡ„ж–№ејҸпјҢз”ўз”ҹзӣ®жЁҷе…§е®№гҖӮ

еҗҢзҗҶжҲ‘еҖ‘д№ҹиғҪеңЁд»Ҙ python еҒҡзӮәзҲ¶йҖІзЁӢзҡ„жғ…жіҒдёӢпјҢд»Ҙ subprocess жЁЎзө„еҚ”еҠ©еҹ·иЎҢеӨ–йғЁжҢҮд»Ө(еӯҗйҖІзЁӢ)гҖҒдёІжҺҘиіҮж–ҷијёеҮә\ијёе…Ҙзҡ„з®ЎйҒ“гҖҒдёҰеҸ–еҫ—иҝ”еӣһеҖјпјҢжң¬ж–Үе°Үд»Ӣзҙ№ subprocess жЁЎзө„з®ЎзҗҶеӯҗйҖІзЁӢзҡ„ж–№жі•иҲҮзЁӢејҸзўјгҖӮ
